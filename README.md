# 🗄️ Архив — Отряд дружественных частиц

Общий репозиторий для хранения и обмена учебными материалами. 

## 📂 Структура
Материалы распределены по соответствующим папкам:
* **Семестр 1**
* **Семестр 2**
* **Семестр 3**

---

## 📥 Инструкция по добавлению файлов
1. Перейдите в нужную папку семестра.
2. Нажмите кнопку **Add file** -> **Upload files**.
3. Выберите файлы на компьютере и подтвердите загрузку кнопкой **Commit changes**.

---

## ⚠️ Правила
* Старайтесь давать файлам понятные названия (например, `Lab1_Physics.pdf` вместо `111.pdf`).
* Не удаляйте чужие файлы без согласования с остальными владельцами.
